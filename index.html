Reference Guide to HTML and CSS Tags

HTML Elements

Table Elements
Tag Name Description
<table> Table The wrapper element that defines a table
<caption> Table Caption Defines a table caption - displayed at top of table, centered
<thead> Table Header Groups the header content in a table
<tbody> Table Body Groups the body content in a table
<th> Table Header Cell Defines a header cell in the table header of a table
<tr> Table Row Defines a row in a table
<td> Table Data Cell Defines a data cell in a table
<tfoot> Table Footer Groups the footer content in a table
<colgroup> Column Group Specifies a group of one or more columns within a table
<col> Column Specifies column properties for each column in a <colgroup> element
Table Attributes
Attributes Belongs to Description
rowspan <td>, <th> Specifies the number of rows a table cell should span
colspan <td>, <th> Specifies the number of columns a table cell should span
